FAQ

What is the best Image Size alternative for iPhone and Mac?
MediaKit, if you want more than resizing. Image Size (by Vitalij Schaefer) is a focused, free ad-supported iOS app for resizing photos to exact pixel, mm, cm, or inch dimensions. MediaKit also resizes to exact dimensions, then adds compression, HEIC/PNG/JPG/WebP/AVIF conversion, EXIF stripping, OCR, and background removal — plus 140+ tools for video, audio, PDF, and archives, all on-device with no ads, on iPhone and Mac via one Universal Purchase.
Can MediaKit resize images to exact dimensions like Image Size?
Yes — MediaKit's Image Resize tool sets exact output dimensions and is one of the five tools that stay free forever. Image Size is excellent at this single job with multitouch crop and px/mm/cm/in units; MediaKit covers exact-dimension resizing and then keeps going with compress, convert, and edit in the same app.
Is MediaKit free like Image Size?
Both have free paths. Image Size is free with ads; you can remove ads for a one-time purchase (~$29.99) and unlock extras like the object eraser separately. MediaKit is free to start with a 3-day full trial, after which five core tools — including Image Resize and Image Compress — stay free forever, ad-free. Pro is $3.99/month, $22.99/year, or $39.99 lifetime.
Does MediaKit show ads or upload my photos?
No on both. MediaKit has no ads and runs 100% on-device via Apple frameworks (CoreImage, Vision, AVFoundation, PDFKit) — photos never leave your device. Image Size also processes on-device, but its free tier is ad-supported.
Why pick MediaKit over Image Size?
Breadth and no ads. If you only resize photos, Image Size is a clean single-purpose tool. If you also compress, convert HEIC, strip EXIF, run OCR, remove backgrounds, or touch video, audio, PDFs, and archives, MediaKit does it all in one ad-free app across iPhone and Mac.
Does MediaKit work on both iPhone and Mac?
Yes — MediaKit is a Universal Purchase: buy once, use on iPhone and every Mac on the same Apple ID. Image Size is also available across iPhone, iPad, and Mac. Pricing and features were verified on 2026-05-31 against the App Store; competitor pricing changes, so re-verify.
